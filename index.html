Speaker Tier Tool — Tech Stack & Maintenance Guide What It Is A single HTML file hosted on GitHub Pages. There is no backend server, no database, and no framework. To update anything, you edit one file on GitHub. Tech Stack GitHub Pages — hosts the tool • URL — https://devinlasker.github.io/speaker-tier/ • File — index.html in the main branch of github.com/devinlasker/speaker-tier • To update — Open index.html in GitHub, click the pencil icon, edit, commit. Changes go live in ~2 minutes. Cloudflare Worker — secure API proxy • Name — speaker-rankings-tool • URL — https://speaker-rankings-tool.dlasker.workers.dev • Purpose — Sits between the tool and the Anthropic API so the API key is never exposed in the browser • To update the API key — Cloudflare dashboard → Workers & Pages → speaker-rankings-tool → Settings → Variables and Secrets → edit ANTHROPIC_API_KEY Anthropic API — powers the AI scoring • Console — console.anthropic.com • Model — Claude Sonnet — uses web search to research each speaker in real time • Cost — Approximately $0.10–0.15 per speaker scored, billed pay-as-you-go • To rotate the key — Create a new key in the Anthropic console, update it in Cloudflare, delete the old one Wikipedia API & Google Trends — free data sources • Wikipedia — Called directly from the browser, no key required. Pulls 5 years of page view data. • Google Trends — Called directly from the browser, no key required. Pulls relative search interest (0–100). If Something Breaks • Tool won't load — Check GitHub Pages is still enabled: github.com/devinlasker/speaker-tier → Settings → Pages • Scoring fails with a network error — Check the Cloudflare Worker is active: dash.cloudflare.com → Workers & Pages → speaker-rankings-tool • API key error — The Anthropic API key may be expired or over budget. Log into console.anthropic.com to check, then update the key in Cloudflare. • Score seems wrong — The scoring rules live inside index.html in the AI prompt section. Search for 'CRITERION' to find and edit them.
